# GLITCH//STUDIO v2

Advanced client-side glitch art studio for images and animations. Create stunning chromatic aberrations, pixel sorts, and data corruption effects directly in your browser.

[![Open in Browser](https://img.shields.io/badge/LIVE_DEMO-Open_Studio-00fff9?style=for-the-badge&logo=googlechrome&logoColor=black)](https://wowkdigital.github.io/glitch_studio/)

## 🚀 Overview
**GLITCH//STUDIO** is a powerful tool designed for artists, designers, and net-punks. It allows for complex layering of digital artifacts and glitches, with real-time preview and multi-format export capability.

## ✨ Features
- **Effects Library**: RGB Split, Scanlines, Pixel Sorting, Data Corruption, Neon Burn, VHS Tape, Hologram, and more.
- **Animation Suite**: Create animated glitches with adjustable FPS and loop modes.
- **Multi-Format Export**: 
  - Static: PNG, JPEG, WEBP.
  - Animated: GIF, WebM, or ZIP (individual PNG frames).
- **100% Client-Side**: No image data is ever uploaded to a server. Everything happens in your browser.
- **Premium Aesthetics**: Dark mode, cyberpunk UI with responsive controls.

## 🛠️ Usage
1. **Upload**: Drag & drop or click to upload your image.
2. **Apply Effects**: Click on effects in the library to add them to your processing chain.
3. **Customize**: Adjust parameters for each effect in the chain.
4. **Animate**: Hit the ⟳ ANIMATE button to see it in motion.
5. **Export**: Save your creation as an image or animation.
